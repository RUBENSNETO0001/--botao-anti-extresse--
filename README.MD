# botão interativo para aliviar o estresse

# descrição do projeto
Este projeto é um botão interativo simples que tem como objetivo proporcionar um momento de alívio do estresse. Cada vez que o botão é pressionado, um contador é incrementado e, ao atingir certos marcos, recompensas visuais são exibidas para incentivar o usuário a continuar interagindo com o botão.
# funcionalidades
- Contador de cliques: Cada vez que o botão é pressionado, o contador é incrementado e exibido na tela.
- Recompensas visuais: Ao atingir 100, 200 e 220 cliques, diferentes imagens são exibidas como recompensa para o usuário.
- Estilo visual: O botão e o contador são estilizados para proporcionar uma experiência agradável ao usuário.
# tecnologias utilizadas
- HTML: Estrutura da página e elementos interativos.
- CSS: Estilização do botão, contador e layout geral da página.
- JavaScript: Lógica para incrementar o contador, armazenar o valor no localStorage e
exibir as recompensas visuais com base no número de cliques.
# como usar
1. Clone o repositório para o seu ambiente local.
2. Abra o arquivo `index.html` em um navegador web.
3. Clique no botão "Aperte" para incrementar o contador e ver as recompensas visuais à medida que você atinge os marcos de cliques.
4. O contador é armazenado no localStorage, então mesmo que você feche o navegador, o número de cliques será mantido quando você reabrir a página.
